# Scanner de Portas TCP - Python 🔍

Ferramenta em Python para auditoria de portas TCP abertas em hosts de rede. Desenvolvido para auxiliar em testes de segurança e mapeamento de superfície de ataque em ambientes corporativos.

### **⚠️ Aviso Legal**
Esta ferramenta foi criada para fins educacionais e para auditoria em ambientes autorizados. Não utilize para escanear redes sem permissão explícita. O uso indevido é de responsabilidade do usuário.

### **🚀 Principais Funcionalidades**
- **Scan de porta única ou range**: Verifique portas específicas ou um intervalo completo 1-65535
- **Multi-threading**: Execução paralela para agilizar escaneamentos extensos
- **Detecção de Banner**: Captura banners de serviços para identificação
- **Exportação de resultados**: Salva o output em `.txt` ou `.csv` para documentação
